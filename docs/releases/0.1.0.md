# dont-deal 0.1.0

## Summary

`dont-deal` 0.1.0 is the first public release of the project.

This release establishes two foundations:

1. a local-first analysis layer that reads time, host context, and git timing data
2. a conservative skill package that helps narrow a chest-pain conversation to two outcomes:
   emergency help now, or urgent same-day medical care

## What is included

- local fatigue snapshot generation
- host detection for Codex / Claude Code / OpenClaw-style environments
- interactive quick-triage CLI
- local-only configuration and history storage
- self-contained `dont-deal-triage` skill package
- successful publication to ClawHub as `dont-deal-triage@0.1.0`

## Important behavior

- this release is intentionally conservative
- it is not a medical diagnosis tool
- it does not modify git history or code
- it keeps health-related context local by default
- first run is bilingual, then it narrows to the user's detected language

## Install notes

Repository:

```bash
git clone git@github.com:shaoqing404/dont-deal.git
cd dont-deal
```

Quick mode:

```bash
DONT_DEAL_HOME=./data npm run triage:quick
```

Self-contained skill package:

```bash
cd skills/dont-deal-triage
DONT_DEAL_HOME=../../data npm run triage:quick
```
